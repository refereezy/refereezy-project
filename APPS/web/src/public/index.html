Bienvenido a Refereezy

Refereezy es una plataforma integral diseñada para la gestión de partidos de fútbol, proporcionando herramientas para árbitros, equipos y organizadores de competiciones.

Nuestro sistema conecta dispositivos móviles con smartwatches para registrar incidentes de partido en tiempo real, garantizando informes precisos y eficientes.

Características Principales:

  • Aplicaciones Móviles para Árbitros

    Aplicación Android que permite a los árbitros gestionar partidos y registrar incidentes de forma rápida y eficiente.

  • Integración con Smartwatch

    Registra incidentes en tiempo real utilizando una aplicación para smartwatch Android, sincronizada con tu dispositivo móvil.

  • Informes de Partidos

    Genera y accede a informes detallados de partidos con todos los incidentes, puntuaciones e información de jugadores.

  • Gestión de Equipos

    Organiza equipos, jugadores y árbitros desde el panel de administración.

Cómo Empezar

1

Registra tu Smartwatch

Utiliza la Consola de Pruebas para conectar tu smartwatch con nuestro sistema.

2

Empareja con tu Cuenta de Árbitro

Conecta tu ID de árbitro con el smartwatch registrado para establecer una conexión.

3

Gestiona Informes de Partidos

Visualiza y gestiona todos los informes de partidos en la página de Informes de Partidos.
